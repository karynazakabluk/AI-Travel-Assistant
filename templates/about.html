AI Travel Assistant

A web project for travel booking, administration and AI-inspired recommendations

Purpose

The purpose of this system is to help users find, book and manage travel destinations. The system combines booking functionality, user administration, database storage and AI-inspired recommendations.

Technologies

Backend: Python Flask

Frontend: HTML and CSS

Database: SQLite

Version control: Git / GitHub

Security

The system uses login, sessions and role-based access control. Users can only see and manage their own bookings, while administrators can manage all bookings.

AI Assistant

The assistant analyzes user input with keyword matching and scoring. It recommends destinations based on region, price level and travel type.
