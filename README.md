# AI Travel Assistant

## Beskrivelse

AI Travel Assistant er en webapplikasjon som hjelper brukere med å finne, bestille og administrere reiser.

Løsningen inneholder:

- Søk etter reiser
- Reisebestilling
- Innlogging og registrering
- Administrasjon av bestillinger
- AI-inspirert reiseassistent
- SQLite-database for lagring av data

## Teknologier

Prosjektet er utviklet med:

- Python
- Flask
- SQLite
- HTML
- CSS
- Git og GitHub

## Formål

Målet med prosjektet er å demonstrere kompetanse innen:

### Utvikling

- Backend-utvikling med Python og Flask

## Funksjoner

### Bruker
